# LifeNavigator

## Dev Setup

Requirements:

+ Node.js
+ Yarn

Clone this repo:

```
$ git clone https://github.com/lancatlin/life-navigator-app.git
```

Install requirements:

```
$ yarn install
```

Start development server (JSON server):

```
$ yarn run db
```

Config your LAN IP to the .env

```
$ vim .env

# use your LAN ip
API_URL=http://192.168.42.174:3000
```

Start Expo

```
yarn start
```
